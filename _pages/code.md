## My-RPG-Game

本项目是在学习 **C++** 和 **面向对象程序设计（OOP）** 过程中完成的练习项目，重点在于 **接口抽象与系统结构设计** ，而非具体游戏内容。

- 实用抽象基类定义核心行为接口
- 通过继承与多态实现功能扩展
- 关注模块解耦与后续可扩展性
- 作为中大型 C++ 项目的结构化实践示例
